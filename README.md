# Kevin van der Toorn - Portfolio Website

Portfolio website built with React, internationalization, and responsive design.

**Live site:** [kevinvandertoorn.com](https://kevinvandertoorn.com)

## 🛠️ Tech Stack

- ⚛️ **React 18** - Modern functional components and hooks
- 🏪 **Redux Toolkit** - State management
- 💅 **Styled Components** - Component-based styling with theming
- 🌍 **i18next** - Internationalization (EN, NL, DE + more)
- ⚡ **Vite** - Fast build tool and development server
- 📱 **Responsive Design** - Cross-device optimization

## ✨ Key Features

- 🌐 **Multi-language Support** - 7 languages with browser detection
- 🎨 **Dark/Light Theme** - Automatic system preference detection
- 🔍 **SEO Optimized** - Meta tags, Open Graph, and structured data
- ⚡ **Performance** - Image preloading and efficient bundling
